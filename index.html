My Project

Here are some overview of the projects that I have developed

Klik Indomaret

This is a large and complex project because it is developing features for Indomaret's e-commerce. In this project I hold 4 sub-projects including food, virtual, travel, tickets.
Klik Indomaret - Food

Klik Indomaret Food is an online food website that users can pick up directly at the Indomaret store or make delivery to their homes. This website serves Indomaret food and beverage products such as point coffee, yummy, washoku sato, mister donut, etc.
Klik Indomaret - Virtual

Klik Indomaret Virtual is a separate website with the Klik Indomaret domain. Users can make virtual payments for credit, PLN, motorbike tax, etc. This website can be accessed directly in the Indomaret store with a touch screen near the cashier or can be accessed directly on the website.
Klik Indomaret - Tiket

Klik Indomaret Tiket is a website that helps users purchase tickets online. Users can buy tickets for rides such as Dufan, Ancol and can purchase concert tickets.
Klik Indomaret - Travel

Klik Indomaret Travel is a website for purchasing travel tickets. Users can buy a plane, train, and ferry tickets. Easy to use in your hand, get priority tickets, and safe in transactions.
SobatSewa

This is a startup that my friends and I created to help in finding rental motorbikes, cars, play stations, futsal fields, etc. This is the place where I explore technology and feel the comparison. I am using ReactJS and VueJS using microservices and learning more about the backend. Because the team is small, I learned a lot about technology, communication and coordination.
Sistem Informasi Sekolah Tumbuh

It is my full-stack project. I worked on the database using MySQL, the back-end using the PHP Codeigniter framework, and the front-end using the Bootstrap framework. This project functions to accept registration between levels starting from elementary - middle - high school at the Sekolah Tumbuh Yogyakarta. I am multi-role and authenticate by logging in and registering using email and monthly reporting into an Excel file.
QR Project - Kayuwangi

The QR Code project is a project to challenge myself in answering social problems. Kayuwangi is a wedding organizer team and I offer to create technology to answer the problem of guests who ask for lots of photos. The QR Code project is a project that I created to generate a QR code for every photo of guests who come so that the guest can download by scanning the QR code. I use Python to create the program.
